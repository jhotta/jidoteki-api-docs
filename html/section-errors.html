Errors and Status Codes

Below you will find a description of the JSON-formatted response for each API request.

Attributes
Name Description
type The type of message returned.
Example: success or error
message A short description of the error.
Example: Unable to authenticate
content This contains additional information regarding the response, such as an access token.
Example: 2e0a696ea8746b5daa6e

HTTP Status Code Summary

  • 200 OK - Success!
  • 201 Created - The request was queued, but not processed yet.
  • 400 Bad Request - You're sending invalid data to the API.
  • 403 Forbidden - Your request was not authenticated correctly.
  • 404 Not Found - The requested URL could not be found.
  • 500 Internal Server Error - A server error which is our fault. Sorry.
